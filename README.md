# Kanban Board (React)

A Kanban-style task management application built using React JS.  
This project allows users to create, manage, and organize tasks across multiple columns using drag-and-drop interactions.

## Features

- Task columns: To Do, In Progress, Done
- Add, edit, and delete tasks
- Drag and drop tasks between columns
- Task details modal with inline editing
- Persistent data using localStorage
- Responsive UI using Tailwind CSS

## Tech Stack

- React JS
- Tailwind CSS
- Context API
- dnd-kit
- localStorage

## Setup

npm install  
npm run dev

## Deployment

The app can be deployed using Versal.
